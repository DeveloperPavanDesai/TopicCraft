# TopicCraft

Build an AI-powered Writing Assistant that:

- Takes a topic
- Accepts a draft written by the user
- Analyzes the content using spaCy
- Provides:
  - Entity and concept extraction
  - Related topic suggestions
  - Style and readability feedback
  - Bias/tone detection
  - Semantic enrichment suggestions

Everything is wrapped inside a Streamlit demo app.

## How This Project Helps (Real-World Value)

**For Writers**

- Suggests missing subtopics
- Improves clarity
- Reduces repetition
- Detects biased or overly emotional tone

**For Businesses**

- Standardizes content tone
- Ensures neutral communication
- Improves SEO keyword richness
